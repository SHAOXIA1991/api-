## 📖 NoFX 机器人：AI 模型添加指南

拿到卡密后，请按照下方步骤完成配置，即可启动 **AI-Grid 智能调价策略**。

---

## ⚙️ NoFX / AI 机器人配置详细步骤

请在你的机器人后台 **「添加 AI 模型」** 窗口中，按照以下参数填写：

---

### ✅ 1. 基础信息填写

| 参数 | 填写内容 |
|------|------------|
| **API Key** | 填入你购买后获得的卡密 |
| **Base URL** | `https://juzi.xstx.info/v1` |
| **Model Name（可选）** | 从下方推荐模型列表中选择填写（建议手动输入，避免拼写错误） |

> ⚠️ **重要提示**  
> Base URL **必须以 `/v1` 结尾**，否则机器人可能无法正确识别接口路径。

---

### ✅ 2. 推荐支持模型列表（直接复制使用）

为了获得最佳交易逻辑与响应速度，建议优先使用以下模型：

| 模型类型 | 建议填写的 Model Name | 特点说明 |
|------------|------------------------|------------|
| 🚀 旗舰逻辑 | `claude-opus-4-5-20251101` | 智商最高，适合复杂行情深度分析 |
| ⭐ 性价比之王（强烈推荐） | `claude-sonnet-4-5-20250929` | 速度与逻辑的完美平衡，刷量首选 |
| ⚡ 极速响应 | `claude-haiku-4-5-20251001` | 响应最快，适合极高频简单挂单策略 |
| 🌐 Google 系 | `gemini-2.5-pro` 或 `gemini-3-pro-preview` | 擅长长文本处理，适合多市场关联分析 |

### 🧠 进阶使用小贴士

#### 🔹 Thinking 思考模型

如果你填写：

```text
claude-sonnet-4-5-20250929-thinking
```

模型会在给出调价建议前进行内部 **“深度思考”**：

- ✅ 策略更稳健  
- ⚠️ 会略微增加 Token 消耗  
- 🎯 适合中低频、高质量决策场景

## ❌ 常见报错排查

| 错误提示 | 可能原因 | 解决方案 |
|-------------|-------------|-------------|
| **404** | Base URL 缺少 `/v1` | 确认链接结尾为 `/v1` |
| **401** | API Key 包含多余空格 | 重新复制粘贴，确保无空格 |
| **Insufficient Balance** | 调用额度已耗尽 | 联系管理员充值 |
